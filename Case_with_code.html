Case Study: Building a Pipeline for Selecting Wines for Wine Distributors/Merchants¶

In this part we'll combine our learnings and results from EDA and Model to try and build a model which could be used by merchants to select a portfolio of wines based on their chemical properties. The modle would optimize for:

  • quality, based on a rating available for similar wines
  • variety, we need to ensure that offered selection varies sufficiently {{to offer enough choice for customers who have different tastes}}

Workflow:

  1. User chooses the number(N) of wine brands to be selected and provides the model a dataset with chemical properties: 1.1. Use the best clustering algorithm and parameters we have found while performing an EDA to generate X clusters (C). 1.2. Using the best ML pipeline (which has the highest 'Recall' score when predicting the lowest quality wines) to select wines which have the highest probability to be high quality from each cluster.
  2. System provides a list of wines which are likely to have a quality rating higher than 6 while ensuring the the selection is sufficient varied.

source_df = utils.load_data()
scaler_preproc = preprocessing.ScalerPreprocessor.factory()()
source_df_std = scaler_preproc.process(utils.clean_df(source_df, "quality"), "quality")

# PCA
binner_preproc = preprocessing.PCAPreprocessor.factory(n_components=11)()
pca_df = binner_preproc.process(utils.clean_df(source_df_std, "quality"), "quality")
pca_df["quality_bin"] = pca_df["quality"].map(utils.bin_quality)

# t-SNE
df_t_sne, _t_sne = preprocessing.TSNEPreprocessor.factory(n_components=2)().process(
    source_df_std, target_var="quality"
)
df_t_sne["quality_bin"] = df_t_sne["quality"].map(utils.bin_quality)

pca_df_3_comp = pca_df[["PC1", "PC2", "PC3"]]

Z = linkage(pca_df_3_comp, method=cluster_settings["method"])

clusters = fcluster(Z, cluster_settings["cutoff"], criterion="distance")

df_reduced_tsne_cluster = df_t_sne.copy()
df_reduced_tsne_cluster["cluster"] = clusters

pca_df_cluster = pca_df.copy()
pca_df_cluster["cluster"] = clusters

source_df_cluster = source_df.copy()
source_df_cluster["cluster"] = clusters
